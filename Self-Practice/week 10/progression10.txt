อาทิตย์นี้คือการรวบรวมทุกอย่างที่เรียนมาให้ครบ ทั้ง HTML, CSS, JS ทำงานด้วยกันเป็นโปรเจกต์เต็ม ๆ ไฟล์ที่ชื่อ Proj ก็
คือโปรเจกต์หลักที่ต้องทำในอาทิตย์นี้ ลักษณะงานก็คือ mini project ที่ดูเป็นเว็บจริง ๆ แล้ว อาจมีฟีเจอร์ซับซ้อนขึ้น เช่น 
ใช้ array method มาจัดการข้อมูลเยอะ ๆ ใช้ event หลายรูปแบบพร้อมกัน หรือบางทีอาจได้ลอง fetch API ดึงข้อมูล
จากไฟล์/เว็บอื่นเข้ามา จุดนี้คือการสรุปความรู้ทั้งหมดใน 4 สัปดาห์ก่อนหน้า แล้วเอามาสร้างผลงานที่ใช้งานได้จริง
